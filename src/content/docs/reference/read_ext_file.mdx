## Description

Reads ext file

## Usage

```r
read_ext_file(
  path,
  line_prefixes = NULL,
  parameters_only = FALSE,
  only_method = NULL,
  only_last = TRUE
)
```

<tr>

<th>Description</th>

<tr>
<td><code>path</code></td>
<td>path to model file, model output directory, ext file or metadata json file.</td>
</tr>
<tr>
<td><code>line_prefixes</code></td>

<td>character, filter for getting estimates from specified method only</td>
</tr>
<tr>

<td>boolean, for grabbing only last estimation method parameters</td>
</tr>

data.frame of ext file
