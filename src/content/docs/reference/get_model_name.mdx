## Description

Extracts the model name from a hyperion_nonmem_model object.

## Usage

```r
get_model_name(model)
```

## Arguments

<div class="arg-table">
<table>
<thead>
<tr>
<th>Argument</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>model</code></td>
<td>A hyperion_nonmem_model object</td>
</tr>
</tbody>
</table>
</div>

## Value

Character string with the model name (filename without extension)

## Examples

```r
## Not run: 
mod <- read_model("models/run001.mod")
get_model_name(mod) # "run001"

## End(Not run)
```
