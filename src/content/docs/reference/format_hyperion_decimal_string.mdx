## Description

Formats numeric values as character strings with a fixed number of decimal places. If `decimals` is `NULL`, falls back to significant figure formatting via `format_hyperion_sigfig_string()`.

## Usage

```r
format_hyperion_decimal_string(x, decimals)
```

<tr>
<th>Argument</th>

<tr>
<td><code>x</code></td>
<td>Numeric value(s) to format.</td>
</tr>
<tr>
<td><code>decimals</code></td>
<td>Number of decimal places. If <code>NULL</code>, uses significant figure formatting instead.</td>
</tr>

## Value

Character vector of formatted numbers. Returns `NA_character_` for `NA` input values.

## See Also

[`format_hyperion_sigfig_string()`](../format_hyperion_sigfig_string/)

## Examples

```r
format_hyperion_decimal_string(0.123456789, decimals = 2)
format_hyperion_decimal_string(c(1.5, 2.567, NA), decimals = 3)

# Falls back to sigfig formatting when decimals is NULL
format_hyperion_decimal_string(0.123456789, decimals = NULL)
```

### Output

```
[1] "0.12" 
[1] "1.500" "2.567" NA      
[1] "0.1235" 
```
