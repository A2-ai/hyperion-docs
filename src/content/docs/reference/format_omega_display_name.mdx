## Description

Builds a display name for omega parameters by appending associated theta information, but only if that information isn't already present in the name. This prevents duplication like "IIV-CL (CL)" when the omega was already renamed to include the theta.

## Usage

```r
format_omega_display_name(name, associated_theta, theta_labels = NULL)
```

## Value

The formatted display name with theta info appended only if missing

## Examples

```r
# Theta already in name - no duplication
format_omega_display_name("IIV-CL", "CL")
# Returns: "IIV-CL"

# Theta not in name - appends it
format_omega_display_name("IIV", "CL")
# Returns: "IIV CL"

# Multiple thetas
format_omega_display_name("IIV", c("CL", "V"))
# Returns: "IIV CL, V"

# With custom labels
format_omega_display_name("IIV", "CL", c(CL = "Clearance"))
# Returns: "IIV Clearance"
```

### Output

```
[1] "IIV-CL" 
[1] "IIV CL" 
[1] "IIV CL, V" 
[1] "IIV Clearance" 
```
