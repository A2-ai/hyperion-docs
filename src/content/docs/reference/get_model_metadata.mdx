## Description

Loads metadata for a NONMEM model from the model's sibling `⁠&lt;model_stem&gt;_metadata.json⁠` file.

## Usage

```r
get_model_metadata(model)
```

## Arguments

<div class="arg-table">
<table>
<thead>
<tr>
<th>Argument</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>model</code></td>
<td>A hyperion_nonmem_model object from <code>read_model()</code></td>
</tr>
</tbody>
</table>
</div>

## Value

hyperion_model_metadata object containing:

- description - Model description string

- tags - Character vector of metadata tags

- based_on - Character vector of parent model references

## Examples

```r
## Not run: 
mod <- read_model("model/nonmem/run001.mod")
meta <- get_model_metadata(mod)
meta$description
meta$tags
meta$based_on

## End(Not run)
```
