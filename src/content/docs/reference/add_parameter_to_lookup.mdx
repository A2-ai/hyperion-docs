## Description

Adds or updates a parameter entry in a TOML lookup file. The parameterization value is validated against known types.

## Usage

```r
add_parameter_to_lookup(
  path,
  name,
  display = NULL,
  desc = NULL,
  unit = NULL,
  parameterization = NULL,
  overwrite = FALSE
)
```
