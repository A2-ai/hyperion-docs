## Description

Holds all parameter comments for a model organized by parameter type (theta, omega, sigma) and validates cross-references between them.

## Usage

```r
ModelComments(theta = list(), omega = list(), sigma = list())
```

## Properties

- **theta**: Named list of ThetaComment objects, keyed by NONMEM name.

 

- **omega**: Named list of OmegaComment objects, keyed by NONMEM name.

 

- **sigma**: Named list of SigmaComment objects, keyed by NONMEM name.
