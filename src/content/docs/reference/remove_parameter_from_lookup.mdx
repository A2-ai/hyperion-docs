## Description

Removes a parameter entry from a TOML lookup file.

## Usage

```r
remove_parameter_from_lookup(path, name)
```

## Arguments

<div class="arg-table">
<table>
<thead>
<tr>
<th>Argument</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>path</code></td>
<td>Path to the lookup.toml file</td>
</tr>
<tr>
<td><code>name</code></td>
<td>Parameter name to remove</td>
</tr>
</tbody>
</table>
</div>

## Value

Invisibly returns the path to the lookup file
