## Description

[![\[Deprecated\]](https://lifecycle.r-lib.org/articles/figures/lifecycle-deprecated.svg)](https://lifecycle.r-lib.org/articles/stages.html#deprecated)

`get_model_summary()` was deprecated in hyperion 0.2.0 in favor of `summary(model)`. It was removed in hyperion 0.3.0.

## Usage

```r
get_model_summary(directory, hide_off_diagonal_params = FALSE)
```
