## Description

Get a model's ancestors






## Usage

```r
get_model_ancestors(mod)
```

## Arguments

<div class="arg-table">
<table>
<thead>
<tr><th>Argument</th><th>Description</th></tr>
</thead>
<tbody>
<tr>
<td><code>mod</code></td>
<td>A `hyperion_nonmem_model` object or a path to a `.mod`/`.ctl` file.</td>
</tr>
</tbody>
</table>
</div>


## Value

Character vector of ancestor project-relative paths (with
extension, e.g., `"models/onecmt/run001.mod"`). Includes `mod` itself
alongside its ancestors. Returns empty vector if the lineage has no
ancestors.
