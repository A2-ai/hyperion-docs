## Description

Creates ETA labels in the format "ETA1//ETA-CL" from diagonal omega parameters. The label uses the associated_theta name if available, otherwise the omega name.

## Usage

```r
get_eta_labels(model_comments)
```

## Value

Character vector of ETA labels (e.g., c("ETA1//ETA-CL", "ETA2//ETA-V"))
