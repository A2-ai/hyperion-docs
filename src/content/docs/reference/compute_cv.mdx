## Description

Calculates the CV% for Omega/Sigma diagonal parameters based on the specified transformation. For LogNormal and AddErr transforms, uses `sqrt(exp(estimate) - 1) * 100`. For Proportional, uses `sqrt(estimate) * 100`. Returns NA for Theta parameters or Identity transform as CV is not meaningful.

## Usage

```r
compute_cv(estimate, param_type, transform = "identity")
```

## Value

CV as a percentage (vector), or NA if not applicable
