## Description

Gets ETA shrinkage metrics from .shk file

## Usage

```r
get_eta_shrinkage(path)
```

## Arguments

<div class="arg-table">
<table>
<thead>
<tr>
<th>Argument</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>path</code></td>
<td>path to model file, model output directory, shk file or metadata json file, or a hyperion_nonmem_model object</td>
</tr>
</tbody>
</table>
</div>

## Value

data.frame of ETA shrinkage metrics

## Examples

```r
## Not run: 
get_eta_shrinkage("model/nonmem/run001/run001.shk")
model <- read_model("model/nonmem/run001.mod")
get_eta_shrinkage(model)

## End(Not run)
```
