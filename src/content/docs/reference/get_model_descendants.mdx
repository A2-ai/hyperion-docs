## Description

Get a model's descendants






## Usage

```r
get_model_descendants(mod)
```

## Arguments

<div class="arg-table">
<table>
<thead>
<tr><th>Argument</th><th>Description</th></tr>
</thead>
<tbody>
<tr>
<td><code>mod</code></td>
<td>A `hyperion_nonmem_model` object or a path to a `.mod`/`.ctl` file.</td>
</tr>
</tbody>
</table>
</div>


## Value

Character vector of descendant project-relative paths (with
extension, e.g., `"models/onecmt/run002.mod"`). Does not include
`mod` itself.
