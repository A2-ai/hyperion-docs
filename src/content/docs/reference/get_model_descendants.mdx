## Description

Find all models whose based_on chain includes the given model.

## Usage

```r
get_model_descendants(lineage, model_name)
```

## Arguments

<div class="arg-table">
<table>
<thead>
<tr>
<th>Argument</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>lineage</code></td>
<td>A hyperion_nonmem_tree object from <code>get_model_lineage()</code></td>
</tr>
<tr>
<td><code>model_name</code></td>
<td>Character, model name (e.g., "run001" or "run001.mod")</td>
</tr>
</tbody>
</table>
</div>

## Value

Character vector of descendant names (without .mod suffix)
