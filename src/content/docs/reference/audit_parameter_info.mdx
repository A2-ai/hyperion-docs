## Description

Shows where each piece of parameter metadata originated (mod file path, lookup file path, "default", or "user supplied").

## Usage

```r
audit_parameter_info(info)
```

</tr>
</tbody>
</table>

## Value

List with theta, omega, sigma data frames showing source paths for each field
