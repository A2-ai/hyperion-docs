## Description

Returns TRUE if model1 is an ancestor of model2 or vice versa (i.e., they are in a direct parent-child chain).

## Usage

```r
are_models_in_lineage(lineage, model1, model2)
```

## Value

Logical, TRUE if models are in direct lineage
