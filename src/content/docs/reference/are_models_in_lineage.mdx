## Description

Returns TRUE if `m1` is an ancestor of `m2` or vice versa (i.e., they
are in a direct parent-child chain).






## Usage

```r
are_models_in_lineage(m1, m2)
```

## Arguments

<div class="arg-table">
<table>
<thead>
<tr><th>Argument</th><th>Description</th></tr>
</thead>
<tbody>
<tr>
<td><code>m1</code></td>
<td>A `hyperion_nonmem_model` object or a path to a `.mod`/`.ctl` file.</td>
</tr>
<tr>
<td><code>m2</code></td>
<td>A `hyperion_nonmem_model` object or a path to a `.mod`/`.ctl` file.</td>
</tr>
</tbody>
</table>
</div>


## Value

Logical, TRUE if models are in direct lineage.
